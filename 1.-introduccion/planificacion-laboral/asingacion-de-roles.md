# Asingacion de roles

* Desarrollador: responsable de implementar las aplicaciones móviles y de escritorio, integración con API REST y manejo de base de datos. (YO)
* Diseñador: encargado de los mockups, diseño UI/UX adaptable y accesible. (YO)
* Técnico de marketing: para difusión, estrategias de captación del cliente y soporte de comunicación.
* Proveedor de hardware/software: suministra equipos, tablets, servidores y licencias.
* Cliente: usuario final de la plataforma web o móvil.
